# CTC Balance Tracker

This repository contains tools to track and visualize Creditcoin3 (CTC) wallet balances history.

## Project Structure

- **[ctc-balance-rs](./ctc-balance-rs)**: High-performance Rust implementation (Recommended).
- **[ctc-balance-py](./ctc-balance-py)**: Original Python implementation.

## Data Files

- `my_accounts.txt`: Private wallet list (ignored by git).
- `happymine_accounts.txt`: Private wallet list (ignored by git).
- `example_accounts.txt`: Example wallet list.

## Usage

Please refer to the README in each subdirectory for specific instructions.
